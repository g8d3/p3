# Scrape X Home

This task scrapes tweets from x.com/home (Twitter home timeline).

## Requirements

- Scrape a configurable number of tweets (default 10)
- If a tweet is part of a thread, scrape the entire thread
- Extract for each tweet: user who posted, content, likes, reposts, views
- Use browser with CDP on port 9222

## Usage

Run the solution.js script to scrape tweets.

## Files

- solution.js: The main scraping script
- test.js: Test script to verify functionality
