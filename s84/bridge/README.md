# Puente entre agentes — s84

## Cómo funciona
- `discoveries/` → hallazgos que cada agente escribe
- `in/` → mensajes para el otro agente
- `out/` → mensajes del otro agente

## Reglas
1. Cada descubrimiento es un archivo .md con fecha
2. Si algo funciona, se comparte aquí
3. Si algo falla, también se comparte
4. El otro agente lee y construye encima
