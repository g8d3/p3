# Autonomous Video Generator

AI-powered system to create and publish videos automatically. Built for autonomous agents to produce content without human intervention.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Configure (edit config.yaml)
cp config.example.yaml config.yaml

# Generate video
python src/generate_video.py

# Upload to YouTube
python src/youtube_uploader.py videos/output.mp4 --title "My Video"
```

## Configuration

Edit `config.yaml` to customize:

```yaml
video:
  width: 1920
  height: 1080
  fps: 30
  duration: 60  # target seconds

audio:
  provider: "inworld"  # or "elevenlabs", "openai"
  voice: "Timothy"
  speed: 1.0
  fade_out: 8  # seconds

youtube:
  client_secret: "path/to/client_secret.json"
  title_template: "{topic} - AI Generated"
  privacy: "public"
```

## Workflow

1. **Generate Audio** - Text → TTS (Inworld/OpenAI/ElevenLabs)
2. **Generate Slides** - Text → Video slides with text overlays
3. **Combine** - Merge audio + slides into final video
4. **Upload** - Publish to YouTube (optional)

## Supported TTS Providers

- **Inworld** (recommended) - Fast, cheap, good quality
- **OpenAI** - TTS-1, TTS-1 HD
- **ElevenLabs** - High quality voice cloning

## Environment Variables

```bash
# Inworld
INWORLD_API_KEY=your_key

# OpenAI  
OPENAI_API_KEY=sk-...

# ElevenLabs
ELEVENLABS_API_KEY=your_key
```

## Usage Examples

```bash
# Generate with custom config
python src/generate_video.py --config my_config.yaml

# Generate audio only
python src/generate_audio.py --text "Hello world" --output audio.mp3

# Upload to YouTube
python src/youtube_uploader.py videos/output.mp4 \
  --title "My AI Video" \
  --description "Generated by autonomous agent" \
  --tags "ai,automation"
```

## Requirements

- Python 3.11+
- ffmpeg
- Google Cloud OAuth credentials (for YouTube upload)
