## Herramientas
Remotion + FFmpeg + MoviePy + TTS Chutes

## Estilos que produce
- Cinematic (24fps, letterbox, grano, teal/orange)
- Documental (colores naturales, transiciones suaves)
- Vaporwave (neón, glitch, retro)
- Kinetic typography (texto animado)

## Output
`output/renders/video_a_*.mp4`

## Cuándo usar este grupo
- Cuando quieres máxima calidad visual
- Cuando el video necesita efectos programáticos complejos
- Cuando tienes tiempo para renderizar (Remotion tarda más)
