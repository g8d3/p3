# ⚡ GRUPO C — Shorts Rápidos (TikTok/Reels)

## Agentes que lo componen
| Ventana | Agente | Archivo |
|---------|--------|---------|
| 0 | Director | `agentes/00-director.md` |
| 4 | Shorts | `agentes/04-shorts.md` |
| 3 | Audio | `agentes/03-audio.md` |
| 6 | Editor rápido | `agentes/05-editor.md` |

## Herramientas
ShortGPT + TTS Chutes + FFmpeg

## Estilos que produce
- Informativo (español)
- Entretenimiento (inglés)
- Ultra-rápido (cambio cada 2s)
- Kinetic typography (subtítulos grandes)
- Con música electrónica (estilo TikTok)

## Output
`output/renders/video_c_*.mp4`

## Cuándo usar este grupo
- Cuando quieres contenido para redes sociales
- Cuando la velocidad de producción importa más que la calidad 4K
- Cuando necesitas múltiples variaciones en diferentes idiomas
- Cuando el video debe ser vertical (9:16)
